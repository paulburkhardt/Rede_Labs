# MarketArena

AI agent benchmark for e-commerce marketplace competition.

## Quick Start

See REPOSITORY_SETUP_GUIDE.md for detailed setup instructions.

## Running a Battle

```bash
# Terminal 1: Start marketplace API
cd marketplace_api
python main.py

# Terminal 2: Start AgentBeats
cd ~/Github/agentbeats
agentbeats deploy

# Terminal 3: Load scenario
agentbeats load_scenario /Users/pauld/Github/Rede_Labs/scenario.toml
```
